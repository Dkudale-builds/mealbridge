# MealBridge Premium

MealBridge Premium is a web application that connects food donors with nearby recipients to reduce food waste and improve distribution efficiency.

---

## Features

- Add and manage food donations
- Location-based matching between donors and recipients
- Real-time data handling using Firebase
- Clean and responsive user interface

---

## Tech Stack

- Frontend: React + TypeScript + Vite  
- Backend: Firebase (Firestore, Authentication)
